== Description ==
Woocommerce extension for adding custom picture mounts and picture frames to product images and thumbnails.

Site admin uploads mount and frame images in WordPress admin pages. Products on shop front-end can then be viewed with any combination of mount and frame.

== Installation ==
1. Upload \\\"woocommerce-picture-framer\\\" directory to the \\\"/wp-content/plugins/\\\" directory.
2. Activate the plugin through the \\\"Plugins\\\" menu in WordPress.
3. Place \\\"do_action( \\\'plugin_name_hook\\\' );\\\" in your templates.
